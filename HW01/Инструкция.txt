## Установка и запуск

### 1. Клонирование/скачивание проекта

Скопируйте файлы проекта в любую директорию.

### 2. Установка зависимостей

Откройте терминал в директории проекта и выполните:

```bash
pip install -r requirements.txt
```

Это установит:
- FastAPI
- Uvicorn (ASGI-сервер)
- Дополнительные зависимости

### 3. Запуск бэкенда

Выполните команду:

```bash
uvicorn main:app --reload --port 8000
```

Сервер запустится на `http://localhost:8000`.

**Проверка работы API:**
- Откройте в браузере `http://localhost:8000/questions` – должен вернуться JSON с вопросами.

### 4. Запуск фронтенда

Откройте файл `index.html` в браузере одним из способов:

- **Двойной клик** по файлу в проводнике
- **Перетащите** файл в окно браузера
- **Через локальный веб-сервер** (рекомендуется для избежания проблем с CORS):
  ```bash
  python -m http.server 8000
  ```
  Затем откройте `http://localhost:8000`

### 5. Использование приложения

1. Страница загрузит вопросы с сервера.
2. Заполните все текстовые поля.
3. Нажмите кнопку **"Отправить ответы"**.
4. После успешной отправки появится зелёное сообщение "Спасибо!".
5. Можно сбросить форму кнопкой **"Сбросить форму"**.
